# 分片集群管理Sharded Cluster Administration

[**Config Server Administration**](https://docs.mongodb.com/manual/administration/sharded-cluster-config-servers/)

**服务器管理配置**

This section contains articles and tutorials related to sharded cluster config server administration

该部分包含了与分片集群配置服务器管理有关的文章和教程

[**View Cluster Configuration**](https://docs.mongodb.com/manual/tutorial/view-sharded-cluster-configuration/)

**查看集群设置**

View status information about the cluster's databases, shards, and chunks.

查看集群数据库，分片和块的状态信息

[**Restart a Sharded Cluster**](https://docs.mongodb.com/manual/tutorial/restart-sharded-cluster/)

**重启一个分片集群**

Stop and restart a sharded cluster.

停止和重启一个分片集群

[**Migrate a Sharded Cluster to Different Hardware**](https://docs.mongodb.com/manual/tutorial/migrate-sharded-cluster-to-new-hardware/)

**把一个分片集群迁移到不同的硬件**

Migrate a sharded cluster to a different hardware system, for example, when moving a pre-production environment to production.

把一个分片集群迁移到不同的硬件到一个不同的硬件系统，例如，移动一个预生产环境到产品中。

[**Add Shards to a Cluster**](https://docs.mongodb.com/manual/tutorial/add-shards-to-shard-cluster/)

**在一个集群中加入分片**

Add a shard to add capacity to a sharded cluster.

通过给一个分片集群添加一个分片以增加其容量。

[**Remove Shards from an Existing Sharded Cluster**](https://docs.mongodb.com/manual/tutorial/remove-shards-from-cluster/)

**从一个现存的分片集群中移除分片**

Migrate a single shard's data and remove the shard.

迁移一个分片的数据并移除分片。

[**Clear** jumbo **Flag**](https://docs.mongodb.com/manual/tutorial/clear-jumbo-flag/)

**清除jumbo标记**

Manually clear jumbo flag from a chunk.

从一个块中手动清除jumbo标记

[**Back Up Cluster Metadata**](https://docs.mongodb.com/manual/tutorial/backup-sharded-cluster-metadata/)

**备份集群元数据**

Create a backup of a sharded cluster's metadata while keeping the cluster operational.

在做集群操作时创建一个分片集群元数据的备份。

[**Convert Sharded Cluster to Replica Set**](https://docs.mongodb.com/manual/tutorial/convert-sharded-cluster-to-replica-set/)

**把分片集群转换成副本集**

Convert a sharded cluster into a single replica set.

把一个分片集群转换成一个副本集。

[**Convert a Replica Set to a Sharded Cluster**](https://docs.mongodb.com/manual/tutorial/convert-replica-set-to-replicated-shard-cluster/)

**把一个副本集转换成一个分片集群**

Convert a replica set to a sharded cluster in which each shard is its own replica set.

把一个副本集转换成一个分片集群，这个集群中的每个分片是其自身的副本集。

[**Convert a Shard Standalone to a Shard Replica Set**](https://docs.mongodb.com/manual/tutorial/convert-shard-standalone-to-shard-replica-set/)

**把一个分片单点转换成一个分片副本集**

Convert a shard standalone instance to a shard replica set.

把一个分片单点实例转换成一个分片副本集。
